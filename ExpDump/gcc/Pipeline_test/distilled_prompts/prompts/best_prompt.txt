- **Assertions and Diagnostics**: <assert.h> for debugging macros.
- **Data Types**: Headers like <stdint.h>, <stdbool.h>, <complex.h> define specific data types and related macros.
- **Character Handling**: <ctype.h>, <wctype.h> for character checks, and <string.h>, <wchar.h> for string operations.
- **Math**: <math.h>, <tgmath.h>, <complex.h> for math functions.
- **Input/Output**: <stdio.h> is for input/output operations.
- **Memory/Utilities**: <stdlib.h> includes memory management and other utilities.
- **Error Handling**: <errno.h> for error codes.
- **Time**: <time.h> handles date/time functions.
- **Localization**: <locale.h> for cultural settings.
- **Concurrency**: <threads.h>, <stdatomic.h> for threads and atomic operations.
- **Variable Arguments**: <stdarg.h> allows functions with variable arguments.
- **System Interactions**: <signal.h>, <setjmp.h> for signals and non-local jumps.
- **Floating-Point**: <float.h>, <fenv.h> define floating-point limits and environment.
- **Integer Limits**: <limits.h> for integer ranges.
- **Unicode**: <uchar.h> for UTF characters.
- **Miscellaneous**: Headers like <iso646.h> (alternative operators), <stdalign.h> (alignment macros), <stdnoreturn.h> (noreturn macro), and others introduced in C11/C23 standards.
